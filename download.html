AntiZyphon

Lightweight scanner for Minecraft mods — detects suspicious classes, keywords and heuristics.

Download (ZIP) Source (GitHub)
Quick & Deep Scan
Choose fast heuristics or more thorough class analysis.
Safe Extraction
Extracts JAR contents into a temporary folder for inspection.
Portable
Single-file ZIP or executable — no installer required.

Install & Run

  1. Download the ZIP and extract it to a folder.
  2. Run the bundled executable or `python index.py` to start the UI.
  3. Place Minecraft mods into the uploads/ folder or use the Upload button in-app.
